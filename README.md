### Install Required Packages
1. Install Pillow
   ```
   pip install Pillow
   ```
2. Install OpenCV
   ```
   pip install opencv-python
   ```
3. Insall Pandas
   ```
   pip install pandas
   ```
4. Insall Matplotlib
   ```
   pip install matplotlib
   ```

    ```
    Requirement already satisfied: [Package] in [Install Path]
    ```
